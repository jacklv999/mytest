2.元素定位

  • static 定位: 即没有定位, 遵循正常的文档流对象

  • relative 定位: 比照 static 定位的元素存在相对位移

    xxxxxxxxxx
    h2.pos_left {position:relative; left:-20px;}
    h2.pos_right{position:relative; left:20px;}

  • fixed 定位: 始终固定在给定位置, 不随窗口滚动而变化, 适用于文章目录等

    xxxxxxxxxx
    p.pos_fixed {position:fixed; top:30px;}  /*始终固定在窗口的给定位置*/

  • absolute 定位: 相对于最近的已定位父元素, 否则相对于页面, 可与其它元素重叠

    xxxxxxxxxx
    h2{position:absolute; left:100px; top:150px;}

  • sticky 定位: 基于用户滚动定位, 类似于 excel 的冻结表格

    xxxxxxxxxx
    div {position: sticky; background-color: green;}

3.Overflow

x
div {overflow: auto;}    /*可选值: hidden(隐藏溢出),scroll(滚动查看溢出),auto(自动)*/

4.Float

  • 浮动参数: float 指定是否可以浮动或浮动方向,
  • 禁止浮动: 使用 clear 参数指定元素周围无浮动元素, 
x
.thumbnail { float:left;}  /*指定浮动方向, 可选值left, right*/
.text_line { clear:both;}  /*指定禁止浮动, 可选值left, right, both*/

5.水平和垂直对齐

  • 水平居中: 使用 margin:auto
  • 垂直居中: 使用 padding: 20px
