## 前缀和

利用前缀和可以计算很多问题，常和hash表结合。

#### 前缀和

```python
## create hash table by dict
dt = {0:-1}
cur = 0

## compute prefix sum
for idx,val in enumerate(nums):
    cur += val
    if cur not in dt:
        dt[cur] = idx
```

在前缀和数组中，可以在 O(1) 时间中计算数组和。

举例来说，对数组`[1,-1,5,-2,3]`，可计算得出其前缀和数组为 `{0: 1, 1: 0, 5: 2, 3: 3, 6: 4}`。所以，和为0的数组为`0-1`; 和为1的数组为`0`; 和为5的数组为 `0-2`。
