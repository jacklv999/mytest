## 双指针

双指针是一种非常常用的技巧，在下面的示例中，利用双指针遍历两个数组，然后依次对比数据。

```python
p_1, p_2 = 0, 0
res = []
while (
    p_1 < len(L1)
) & (
    p_2 < len(L2)
):
    if L1[p_1] == L2[p_2]:
        res.append(L1[p_1])
        p_1 += 1
        p_2 += 1
    elif L1[p_1] > L2[p_2]:
        p_2 += 1
    else:
        p_1 += 1
```
