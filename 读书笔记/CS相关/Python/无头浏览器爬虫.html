2. 实时可视化页面

将实时页面转换为 base64 图片, 然后在jupyter notebook中可视化

xxxxxxxxxx
import base64 
import numpy as np
from PIL import Image
import io
import matplotlib.pyplot as plt
​
tt = driver.get_screenshot_as_base64()
​
​
def stringToRGB(base64_string):
    imgdata = base64.b64decode(
        base64_string
    )
    image = Image.open(
        io.BytesIO(imgdata)
    )
    return np.array(image)
​
plt.imshow(stringToRGB(tt))

6. 下载文件

无头浏览器默认不能下载文件, 需要添加以下代码, 但是因为不能确定下载状态, 所以需要慎重使用.

x
downloadPath = "C:\\Users\\"
​
driver.command_executor._commands[
    "send_command"
] = (
    "POST", 
'/session/$sessionId/chromium/send_command'
)
params = {
    'cmd': 
    'Page.setDownloadBehavior', 
    'params': {
        'behavior': 'allow',
        'downloadPath':
        downloadPath
          }
         }
command_result = driver.execute(
    "send_command", 
    params
)
