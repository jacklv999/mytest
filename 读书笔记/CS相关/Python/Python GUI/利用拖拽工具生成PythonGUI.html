利用拖拽工具生产PythonGUI

PythonGUI通常不够方便, 因为没有合适的工具, 最近我发现PAGE和activetcl 可以用于自动化生产PythonGUI界面.

安装依赖

生成前首先需要安装依赖PAGE和activetcl, 两者均可在官网下载

拖拽生成界面

生成过程类似于VB的界面, 可以直接拖拽并设置属性

然后, 点击Gen_Python生成代码

写Callback函数

然后可以在Python编辑器中写相应的Callback函数, 也可以在PAGE中点击 bindings 编写函数
