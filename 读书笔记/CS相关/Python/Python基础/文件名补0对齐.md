## 文件名补0对齐

有时, 下载的文件名为0-N排序, 但是文件名的位数不一致, 导致部分软件不饿能识别. 

因此, 可以使用Python自动补齐文件名.

以下代码为: 对文件名少于6位的文件, 在前面补零至6位

```python
import os

path =r'.\1'
for file in os.listdir(path):
    name = file.split('.')[0]
    os.rename(os.path.join(path, file), 
              os.path.join(path, 
                           '%06d' % int(name) + 
                           ".png")
             )
```
