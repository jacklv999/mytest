1.使用mermaid制作关系图

声明图形方向

使用 graph 参数确定方向

  • Code

    xxxxxxxxxx
    graph TB                    #可选参数 TB  BT  LR  RL
    ​
    start --> Stop

  • Result

设定节点形状和连线

  • 1.节点形状: 默认形状为直角框

    • Code

      xxxxxxxxxx
      graph LR
      A[直角节点]--- B(圆角节点)--- C((圆形节点))--- D>非对称节点]
      E[/输入框/]--- ff[\输出框\]--- G{菱形节点}--- H{{六边形节点}}

    • Result

  • 1.节点连线

    • Code

      xxxxxxxxxx
      graph TB
      A1 --text--> B1
      A2 ---|text| B2
      A3 -.text.-> B3
      A4 -.-|text| B4
      A5 ==text=== B5
      A6 ==>|text| B6

    • Result

子流程图

  • Code

    xxxxxxxxxx
        graph TB
        subgraph one
            A1 --> B1
        end
        subgraph two
            A2 === B2
        end
           subgraph three
            A3 -.-> B3
        end

  • Result

样式化节点

  • Code

    xxxxxxxxxx
        graph LR
            id1(Start)-->id2(Stop)
            style id1 fill:#f9f,stroke:#333,stroke-width:4px
            style id2 fill:#ccf,stroke:#f66,stroke-dasharray: 5, 5

  • Result

2.使用flow制作流程图

  • Code

    xxxxxxxxxx
    flow
    st=>start: Start
    op=>operation: Your Operation
    cond=>condition: Yes or No?
    e=>end
    ​
    st->op->cond
    cond(yes)->e
    cond(no)->op

  • Result
