## SQL related

#### SQL 

**Getdate**:

return date in `2008-12-29 16:25:46.635`;

**Dateadd**:

func format: `DateAdd(interval, number, date)`;

- interval:
  - yyyy/yy, add by year;
  - q, add by quarter;
  - m, add by month;
  - d, add by day;
  - w/ww, add by week;
  - hh, add by hour;
  - s, add by second;
