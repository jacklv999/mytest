AB Test

  • Abtest如何实施:

    • 需求分析 — 确定用户对网页皮肤的偏好

    • Choosing key indicators;

      • 核心指标: 点击率
      • 辅助指标: 根据用户行为漏斗进行设定；
      • 反向指标: 负面影响相关的指标，如回跳率

    • Decide the user amount of experiment;

      • 计算总体指标, 如果转换率

        • 结合显著性和功效计算所需样本

      • 计算平均指标, 如人均消费

        • 使用两样本t检验计算所需样本

      • 对比不同方案

        • 使用卡方检验

    • A/B test gray roll out: make sure there no unacceptable results;

      • AA Test for data point buried, traffic design
      • 5% traffic AB Test

    • A/B test full roll out;

    • Analysis Result: usually, we would use one/two week data

  • AB Test需要注意的地方

    • 周期性行为, 节假日影响
    • 分组/样本代表性

  • AB试验难点；

    • 实验组和对照组产生关联，使实验组策略「溢出」到对照组

      • 按社交关系给用户画出社交圈，再对圈子分流
      • 对cluster进行分流：所有cluster的中心作为control

    • 长期实验效果衡量

      • 加入与长期效果有因果关系或强相关的短期指标

    • 试验效果异质性

      • HTE分析
