Transformer 笔记

1.Self-Attention机制

  • self-attention的示意图

  • self-attention第一步: 对输入 进行变形, 再乘以三个不同矩阵, 得到 三个变量, 其中 attention 的内在机制为:

    • 将目标字作为 Query , 其上下字作为 Key, 目标字的原始值为 Value

    而 self-attention 对 attention 做了一些改变, 改变后的 self-attention 在 transformer 中的效果更好, 其过程为:

    • 首先, 通过线性变化获得目标字的 表示, 上下各个字的 表示以及目标字与上下文各个字原始 表示
    • 然后, 计算 和 两个向量的相似度作为权重
    • 最后, 加权融合目标字的 向量和上下字的 向量, 作为 Attention 的输出, 即目标字的加强语义表示

  • self-attention第二步: 由 对所有 做 attention, 并使用 softmax 得到

  • self-attention第三步: 使用 和 value 相乘, 得到第一个输出值

3.Transformer编码器

输入信息加上人为设定的 Position 信息后输入编码器, 编码器中第一层为 self-attention layer , 然后第二层是前向神经网络, 但是存在 short-cut, 输入信息 a 经过前向神经网络后再次和 b 相加得到 , 再次经过 layer norm 之后输出.

4.Transformer解码器

在 Transformer 的解码器中, 前一个输出信息和人工设定的位置信息相加后进入第一个 attention layer, 在提取 attention 信息后, 再和原来的信息进行相加并 layer norm 之后, 和输入信息一起进入第二层 attention layer 提取 attention 信息; 第一个 attention layer 的输出会再次相加并输入前向传递层, 然后再次 add&norm 之后通过词向量输出
