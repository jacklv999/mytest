Transformer历史

self-attention

2. Self-supervised learning: iGPT

论文框架如下,

  • 首先, 将图片下采样变为低像素的图片. 然后展开为一维序列;
  • 其次, 将一维序列输入GPT, 并训练N predict和mask predict任务;
  • 最后, 进行fine-tune进行图像分类.

这一模型在大多数任务上取得了SOTA.

3. Image classification: ViT

ViT的改动比较小, 主要是将输入从down sampling的结果改成了原图像切块. 同时加入了Position Embedding.

这一模型同样取得了SOTA. 同时还有其他结论:

大数据下结果会更好: 不断增加数据, Transformer表现增加, 但是传统CNN逐渐停滞

  • 但另一方面, 在小数据集上, transformer表现远差于传统CNN;

4. Object detection: DETR

结合CNN和transformer构建行的OD模型. 在本模型中:

  • Feature extract还是CNN;
  • 取得feature之后, 全部采取transformer进行处理;

但是, 第三步为了输出box, 会随机化一些 object queries, 然后learn ground truth.

缺点: 训练速度很慢;

5. OD: Deformable DETR

为了改善DETR的训练速度, 增加了多个feature层.

对任何一个query, 按照下列的路径开始. 在多层feature中依次进行.

6. Semantic segmentation: SETR

思路同样是对图像进行切块, 然后再进行少量的卷积以进行输出.

7. Connecting text and image

CLIP

连接图文进行零样本的分类. 分为两步:

  • 首先, 对文本和图片编码,, 然后算相似度, 使对角线值最大化;
  • 然后, 对输入的图片编码, 从文本中选取相似度最高的输出;

DALL-E

利用文本进行图片生成.
