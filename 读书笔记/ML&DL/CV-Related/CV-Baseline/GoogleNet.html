2.其他改进

  • 1.加深和加宽网络
  • 2.引入稀疏性, 稀疏卷积网络结构, 即使用多尺度的 Kernel 模拟稀疏网络
  • 3.最后的卷积层后使用全局均值池化层替换了全连接层
  • 4.在最后的Fully Connected Layer 前引入 DropOut Layer

3.Training Methodology

  • 1.不同的网络层在不同尺度的图片训练集上训练, 8%-100%的尺寸
  • 2.对图片进行光度畸变可以缓解过拟合

4.Inception Naive

  • 1.最初的Inception结构, 缺点是参数多, 且连接层过厚
  • 2.各卷积层和Pooling层的连接方式: Concate连接

5.Inception V1

  • 改进1: 使用1x1卷积减少维度，同时修正线性激活（ReLU）
  • 改进2: 减少参数量, 卷积参数量已经减少了约了4倍

6. Inception V2

  • 提出 Batch Normalization 方法, 旨在改善普遍使用SGD时的缺陷

  • BN的优点:

    1. 可以适应较大的 Learning Ratio
    2. Less careful Initialization
    3. Act as regularizer in same cases eliminating DropOut

7.Inception V3

  • 1.Suitably Factorized Convolution

    • 1.用2个连续3x3卷积代替5x5卷积层，可在保持感受野范围的同时减少参数量

    • 2.用3个连续的 3X3 卷积代替 7x7 卷积层

    • 2.通过大量实验表明，其并不会造成表达缺失

    • 3.GoogLeNet团队发现任意nxn卷积都可以通过1xn卷积接nx1卷积替代

    • 4.但网络前期这种分解效果并不好，在中等特征图上使用效果更好（特征图大小建议在12到20之间）
