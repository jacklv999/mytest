1. Decision Tree

DT is a if-then-else besed supervised learning algorithm. There are many implement for DT, including:

  • ID3: Creating node based on Information Gain, ergo once we have Feature X, the uncertainty of Feature Y decreased.
  • CART: Recurrently binarization every feature to split the feature space to limit space, then compute probability of every feature
  • Others....

  • Function for compute Entropy of collection;

    def calculate_info_entropy(dataset):
        n = len(dataset)
        # 用Counter统计一下Y的数量
        labels = Counter(dataset[:, -1])
        entropy = 0.0
        # 套用信息熵公式
        for k, v in labels.items():
            prob = v / n
            entropy -= prob * math.log(prob, 2)
        return entropy

  • Function for split data to sub-data;

  • Feature select function: to select the biggest Information Gain feature and then split data by that feature;

    def choose_feature_to_split(dataset):
        n = len(dataset[0])-1
        m = len(dataset)
        # 切分之前的信息熵
        entropy = calculate_info_entropy(dataset)
        bestGain = 0.0
        feature = -1
        for i in range(n):
           # 根据特征i切分
            split_data, _ = split_dataset(dataset, i)
            new_entropy = 0.0
            # 计算切分后的信息熵
            for data in split_data:
                prob = len(data) / m
                new_entropy += prob * calculate_info_entropy(data)
            # 获取信息增益
            gain = entropy - new_entropy
            if gain > bestGain:
                bestGain = gain
                feature = i
        return feature

2. Ensemble Learning

There are many ensemble learning algorithm, including:

  • Bagging

    • Sampling with replacement to create new train set;

    • Train new model with new data set;

    • predicate:

      • classification: vote to predicate;
      • Regression: compute mean;

  • Boosting

    • Train base model on original data;
    • Increasing the weight of misclassificated label in new turn and get new model;
    • repeat to get M model
    • Predicate: the same as before;

  • Stacking

    • Train model based on subset of data;
    • generating new data based on the prediction of unused data in step 1;
    • Repeat!
