3. Related Work

Structured losses for image modeling

Img-to-img translation used to treat the output as “unstructured” which means pixels are considered independently.

The CGAN learn a structured loss in which the loss can penalize any structure that differs from output and target.

Conditional GANs

Other papers had use GAN for img-to-img mappings in unconditional way, relying on others (L2) to get conditional results.
